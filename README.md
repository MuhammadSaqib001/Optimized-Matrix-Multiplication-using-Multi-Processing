# Optimized Matrix Multiplication using Multi-Processing [Parallel & Distributed Computing BS Project]

Welcome to the "Optimized Matrix Multiplication using Multi-Processing" GitHub repository! This project explores and showcases the efficiency of matrix multiplication through optimization techniques and multi-processing, highlighting the performance gains over traditional methods.

## Description

Matrix multiplication is a fundamental operation in linear algebra, widely used in scientific, engineering, and data analysis applications. This repository provides an in-depth study of matrix multiplication optimization, with a focus on the Strassen multiplication algorithm. It covers optimal order determination, the implementation of the Strassen algorithm, and performance evaluation.

Key aspects of this repository include:

1. **Optimal Order of Matrix Multiplication:** Algorithms for finding the optimal multiplication order to reduce computational complexity.
2. **Strassen Multiplication Algorithm:** Implementation and explanation of the Strassen algorithm for improved matrix multiplication.
3. **Performance Evaluation:** Code to measure execution time for both traditional and Strassen multiplication methods.
4. **Multi-Processing and Virtual Machines:** Utilizing multi-processing and VMs to parallelize computation and enhance speed.
